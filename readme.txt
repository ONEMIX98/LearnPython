PYTHON LEARNING NOTES/CODES/PROJECTS

hanoi.py: 汉诺塔游戏。输入数字、三个塔的名字，输出执行步骤；

str2float.py: 字符串转化为浮点数。python中可以用float(str)直接实现，这里利用map/reduce和lambda高度抽象；

prime.py: 输出n以内的素数或者输出n个素数。利用filter实现过滤非素数；

decorator.py: 装饰器练习。 要注意传给装饰器的函数在传入那一刻起就不变了，后来利用装饰器调用
使得产生新的函数，但是在装饰器内部调用的原始函数还是原来那个函数

spdDetect.py, triangle.py, digit.py, calculator.py, bubsort.py:
电子科大暑期编程作业，分别实现速度监测、输出三角形、分解任意整数、两个数简单计算器，冒泡法排序，
详细说明见文件。
